
# SecurityAlgorithm #
SecurityAlgorithm is a Java class library that allow you to encrypt / decrypt a string or a file with many algorithms, such as:

-   AES;
-   Blowfish;
-   DES;
- 	3DES;
-   MD5;
-   RSA;
-   SHA (1, 256, 384, 512).

It also provide an implementation of steganography.
